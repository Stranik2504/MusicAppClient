# MusicApp

MusicApp — это современное Android-приложение для прослушивания музыки, разработанное с использованием последних технологий Android-разработки.

## Особенности

- **Воспроизведение музыки:** Полнофункциональный плеер с поддержкой Media3 (ExoPlayer), очередью воспроизведения и мини-плеером.
- **Поиск:** Поиск треков, альбомов и исполнителей.
- **Библиотека:** Управление вашими плейлистами и сохраненными треками.
- **Профили исполнителей:** Детальная информация об исполнителях и их популярных треках.
- **Детали альбомов/плейлистов:** Просмотр содержимого альбомов и пользовательских плейлистов.
- **Аутентификация:** Система логина и регистрации пользователей.
- **Аккаунт:** Управление профилем пользователя.

## Стек технологий

- **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) — современный инструментарий для создания нативного интерфейса.
- **Навигация:** Navigation Compose для переходов между экранами.
- **Сеть:** [Ktor](https://ktor.io/) — клиент для выполнения HTTP-запросов.
- **Аудио:** [Media3 / ExoPlayer](https://developer.android.com/guide/topics/media/media3) — библиотека для работы с медиа.
- **Локальное хранилище:**
    - **Room:** Для кеширования данных.
    - **DataStore:** Для хранения настроек и токенов сессии.
- **Архитектура:** Чистая архитектура (Clean Architecture) с разделением на слои:
    - `data`: Реализация репозиториев и работа с API/БД.
    - `domain`: Бизнес-логика, модели и Use Case.
    - `presentation`: UI (Compose) и ViewModel.
- **Dependency Injection:** В проекте используется паттерн Service Locator через класс `Creator` для управления зависимостями.
- **Изображения:** [Coil](https://coil-kt.github.io/coil/) для асинхронной загрузки изображений.

## Архитектура проекта

Проект разделен на следующие основные пакеты:

- `dev.stranik.musicapp.data`: Работа с удаленным API (Ktor) и локальной базой данных.
- `dev.stranik.musicapp.domain`: Определения моделей данных и бизнес-сценариев (Use Cases).
- `dev.stranik.musicapp.presentation`: Компоненты интерфейса (Screens, Components) и ViewModel для управления состоянием.
- `dev.stranik.musicapp.navigation`: Настройка графа навигации приложения.

## Установка и запуск

1. Клонируйте репозиторий.
2. Откройте проект в Android Studio.
3. Убедитесь, что у вас установлен Android SDK.
4. Выполните сборку проекта через Gradle.
5. Запустите приложение на эмуляторе или физическом устройстве.
